<p>Proposals that are Stage 1 and above must be transferred to <a href="https://github.com/tc39">the TC39 GitHub organization</a> for discoverability and archival purposes. To onboard a proposal that lives outside the TC39 organization:</p>
<ol>
<li>Transfer your repository to the <a href="http://github.com/tc39-transfer">@tc39-transfer</a> organization</li>
</ol>
<ul>
<li>if you are a TC39 delegate, but not an admin in that organization, please contact <a href="https://github.com/ljharb">@LJHarb</a></li>
</ul>
<ol start="2">
<li><a href="https://github.com/bterlson">@bterlson</a>, <a href="https://github.com/gesa">@gesa</a>, or <a href="https://github.com/codehag">@codehag</a> will transfer your repository to the TC39 organization the next chance they get.</li>
</ol>
<p>Note that as part of the onboarding process your repository name may be normalized. Don't worry, repo redirects will continue to work <strong>as long as</strong> you never create a fork, or a new repository, with the same name - although Github Pages redirects will be broken (please update your links!).</p>
